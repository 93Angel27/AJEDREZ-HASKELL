Para compilarlo debemos simplemente abrir la carpeta "Ajedrez" en Visual Studio para que no haya problemas con el import del módulo y, 
en el fichero del main.hs, darle a Run File. Posiblemente, como ya se ha comentado en la memoria, se debe instalar split de 
Data.List con el siguiente comando: cabal install split. Los ejemplos que hemos subido a la memoria son posiciones mientras se
realiza una partida. Las posiciones que usamos son las típicas del ajedrez.

Nuestra versión es la 8.0.2. Suponemos que si en esta versión (que es anterior) va entonces en la 8.6 irá también.
